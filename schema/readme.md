# HIMSA Dataspace

## Aufgabe

* Erstellen von Ecore Meta-Modellen aus den XSD's
* Erstellen einer REST Schnisstelle für CRUD von Daten
* Persistenz der Daten in MongoDB
* Gecko EMF 6
